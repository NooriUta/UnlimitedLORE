Единый паттерн поиска и фильтров для LORE — прототип

Единый поиск + фильтры · прототип

Один алгоритм на всех вкладках: поиск в одном месте и виде → фильтр-бар из групп-чипов (счётчики faceted, единый стиль, доступны с клавиатуры) → всегда видимая строка активных фильтров (снять по ✕) → Сбросить всё. Меняется только НАБОР групп под вкладку — механика и вид идентичны.

Один и тот же компонент <SearchFilter> на трёх вкладках — разный только конфиг групп. Поиск фильтрует по id/названию; чипы — по фасетам; счётчики пересчитываются под текущий выбор; активные фильтры видны строкой и снимаются по ✕. Клик по чипу = toggle. Так пользователь привыкает к одному поведению везде.
Прототип демонстрационный (mock-данные). Внедрение — после approve, общими примитивами <SearchInput> · <FilterBar> · <Chip> · <ActiveFilters>.
